# Analytics Review SOP

## Checkpoints
We rely on 3 standard checkpoints post-publish to determine the performance of a video. Checkpoints sit in the "Analytics" tab.

### 48-Hour Review
- **Objective:** Measure initial CTR and Hook conversion.
- **Action:** 
  1. Record CTR (Goal > 5%). 
  2. Record 30-sec retention % (Goal > 60%).
  3. If CTR is < 3%, schedule a Thumbnail A/B test via YouTube Studio. Add note in "Packaging Notes".

### 7-Day Review
- **Objective:** Assess sustained search/recommendation velocity.
- **Action:**
  1. Measure daily average view velocity.
  2. Identify top 2 traffic sources. 
  3. Formulate the "Improvement Summary" regarding pacing and narrative dropoffs.

### 28-Day Review
- **Objective:** Calculate overall ROI and set series plans.
- **Action:**
  1. Determine total watch hours.
  2. Record new subscribers.
  3. Finalize "Series Follow-up Suggestion" (Does this topic warrant a Part 2?).
