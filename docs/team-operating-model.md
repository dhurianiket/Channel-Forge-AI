# Team Operating Model

## Overview
This platform supports scaling from a solo creator to a small team without losing editorial control and governance.

## Default Team Configurations

### Solo Mode
- One person owns every stage.
- Batching is strongly encouraged.
- Owner role applies everywhere.

### 2-Person Mode
- **Producer / Editor-in-Chief:** Final authority on script, edit QA, and publish. Handles metadata and publishing.
- **Script/Visual Generalist:** Handles ideation, research, script drafting, and visual asset prep (thumbnails, prompting).

### 3-Person Mode
- **Producer / Editor-in-Chief:** Edit QA, upload, analytics.
- **Script / Researcher:** Topic ideation, script drafting, fact-check prep.
- **Visual / Thumbnail:** Asset generation, prompts, video/image edits.

### 4-Person Mode
- **Producer / Editor-in-Chief:** Output decisions, strategy.
- **Script / Researcher:** Scripting.
- **Visual / Thumbnail:** Assets.
- **Ops / Analytics Manager:** Calendar, upload prep, postmortems, 48h/7d reviews.

## Delegation Logic
The lowest level tasks are assigned first, but final approval runs through the Producer.
