# Template Governance

## Objective
Template internal operations, not public-facing content. We structure templates to maintain quality and minimize repetitive outputs.

## Allowed Templates
- Script Structure Template (e.g. Hooks, 3-act beats).
- Visual Style Guide
- Thumbnail Rules
- Metadata Checklists

## Rule Enforcement
Users must check out or reference these SOPs prior to approving `FACT_CHECK`, `ROUGH_CUT`, or `METADATA`. These templates ensure standard operational practices.

A template can be Workspace-wide or Channel-specific.
